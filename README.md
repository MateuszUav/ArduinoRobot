# ArduinoRobot

## Overview
Arduino-based mobile robot project focused on embedded control, motor driving, and basic autonomous behavior. Integrated with simple python program for communication. 

## Features
- Microcontroller: Arduino (Uno/Nano)
- Companion Computer
- Motor control via H-bridge
- Sensor integration (e.g., ultrasonic / IR / line sensors)
- Basic autonomous logic

## Architecture
- Arduino firmware (C/C++)
- Python interface
- Direct GPIO control
- PWM motor control

## Use Cases
- Line following
- Pid controll

## How to Run
1. Open project in Arduino IDE.
2. Select correct board and COM port.
3. Upload firmware.
